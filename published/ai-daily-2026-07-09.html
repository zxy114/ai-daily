GPT-Live 全双工语音

OpenAI 上线 GPT-Live：全双工语音让 ChatGPT「像同事一样说话」

VentureBeat / OpenAI 官方 · 2026-07-08 · venturebeat.com · openai.com
OpenAI 7 月 8 日发布 GPT-Live 语音双模型（GPT-Live-1 + GPT-Live-1 mini），iOS / Android / ChatGPT.com 同步上线。GPT-Live-1 是付费（Go / Plus / Pro）默认语音模型，mini 服务免费用户。技术核心是「全双工架构」：模型边听边说、连续处理输入输出、可每秒多次决定「说话 / 继续听 / 暂停 / 打断 / 调用工具」，告别 Advanced Voice Mode 时代「沉默即结束」导致的频繁误打断。架构上首次将「语音交互层」与「推理层」解耦——简单问题 GPT-Live 直接回答，复杂任务异步委托给后台 GPT-5.5，期间可继续与用户对话。OpenAI 表示后续将持续把最新前沿模型接入 GPT-Live 后端。
核心事件ChatGPT 语音进入第三代「全双工」时代，OpenAI 重写语音交互栈
关键数据GPT-Live-1 + mini 双模型；全球同步上线；支持多层级付费体系；后台默认 GPT-5.5
影响范围C 端语音助手体验跃迁；企业级实时语音 Agent 落地门槛大幅降低

VentureBeat / Tencent Hunyuan / Hugging Face / Tencent Hy 官网 · 2026-07-08 · venturebeat.com · huggingface.co · hy.tencent.com
