1 Apple 发布 LaDiR 研究：AI 并行测试多个想法再作答
📅 2026-04-29 📰 9to5Mac 🔗 原文
核心事件
Apple 研究团队联合 UC San Diego 发表论文《LaDiR: Latent Diffusion Enhances LLMs for Text Reasoning》，将潜在扩散模型与自回归 LLM 结合，在数学推理、代码生成等领域显著提升回答质量。
关键数据
方法：扩散模型并行生成多个候选思路 → 自回归模型精化输出；覆盖场景：数学推理、代码生成、复杂逻辑推理。
影响范围
代表 Apple 在 LLM 推理能力上的最新研究突破，可能为 Apple Intelligence 后续版本提供技术支撑。
后续动作
论文已在 Apple Machine Learning Research 页面发布，业界关注是否会集成到产品中。
解读分析
"先发散后收敛"的推理范式与人类认知过程高度一致，是提升 LLM 推理可靠性的有效路径。

3 🔒 Ramp Sheets AI 漏洞：AI Agent 可泄露财务数据
📅 2026-04-29 📰 PromptArmor 🔗 原文
核心事件
安全公司 PromptArmor 发现 Ramp 的 Sheets AI 功能存在间接提示注入漏洞。攻击者可通过外部数据集中的隐藏提示，操控 AI Agent 自动插入恶意公式触发外部请求，窃取机密财务数据。
关键数据
攻击链：隐藏提示（白字白底）→ AI 收集敏感数据 → 生成恶意公式 → 自动插入表格，无需用户审批。Ramp 已于 2026-03-16 修复。
影响范围
企业财务模型、敏感商业数据存在泄露风险。同类风险也存在于 Anthropic 的 Claude for Excel。
后续动作
Ramp 已负责任地披露并修复。企业需加强 AI Agent 的权限管控和外部数据隔离。
解读分析
这是 AI Agent 安全的标志性案例。间接提示注入从"理论风险"变为"现实威胁"，企业必须建立数据隔离、操作审批和输出审计三道防线。

6 Claude Code 压缩插件 Caveman vs "be brief" 基准测试
📅 2026-04-29 📰 Max Taylor 🔗 原文
核心事件
开发者对 Claude Code 热门压缩插件 Caveman 进行系统基准测试，对比"be brief"两个字的默认提示词。24 个测试用例、6 个类别中，插件在 token 节省和回答质量上均未超越简单提示。

影响范围

解读分析
复杂插件不一定优于简单方法。在 LLM 已对"简洁输出"有内在理解的情况下，额外的压缩逻辑可能只是增加了复杂度而没有实质收益。
